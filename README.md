# ts-node-starter
Starter kit for Node.js using TypeScript.

---

TypeScript 2.8 compatible.

## Setup
```
$ npm install
```

## Run (src/index.ts)
```
$ npm start
or
$ npm run ts src/index.ts
```

## Build & Run (build/index.js)
```
$ npm run bs build/index.js
```

## Run (watch mode)
```
$ npm run watch
or
$ npm run nd src/index.ts
```

## Test (one-time)
```
$ npm test
```

## Test (watch mode)
```
$ npm run test:w
```
