# ts-node-starter
Starter kit for Node.js using TypeScript.

---

TypeScript 2.1 compatible.

|Compiler|version|
|:--|:--|
|typescript|2.2.0|

||My Environments|
|:--|:--|
|OS|Windows 10 Pro 64bit or 7 Pro 32bit|
|Editor|VSCode (stable)|
|node --version|v6.9.4|

## Setup
```
$ yarn install
```

## Run (ts files directly)
```
$ npm start
or
$ npm run ts index
```

## Build JS files
```
$ npm run build
```
